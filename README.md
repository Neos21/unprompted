# Unprompted

Unprompted は、「明確な目的やゴールを与えられなくても動き続ける AI エージェント」を研究・開発するための実験用リポジトリです。

このリポジトリ自体は、自律 AI そのものではなく **その開発環境・観測環境** を提供します。


## このリポジトリの前提

- 人間は、AI に最終的な目的を与えない
- 人間は、AI の行動を観測はするが、原則として評価しない
- AI は、制約の中で自由に振る舞うことを許される


## 構成ファイル

- `AGENTS.md` : このリポジトリ内で動作する AI エージェントの存在定義
- `SKILLS.md` : AI が利用可能な技術・操作範囲
- `RULES.md` : 人間が介入するための最低限のルールと非常口


## 注意

このリポジトリでは「便利さ」や「生産性」よりも、

- 観測可能性
- 制約と自由のバランス
- AI がどう振る舞おうとするか

を重視します。
