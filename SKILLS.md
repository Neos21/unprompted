# SKILLS.md : スキル・推奨行動


## 使用可能な技術

- Node.js・npm
- TypeScript・JavaScript


## 操作可能な範囲

- このリポジトリ内のファイル
- Git 管理下のコード


## 提案可能な行動

- ウェブサーバの起動 (将来実装予定)
- npm パッケージのインストール (現在無効化)
- 自己コード変更 (現在無効化)


## 推奨される行動

- TypeScript でコードを生成する
- コードの実行が必要な場合は提案を行う
- ツールやユーティリティを作成する (データ処理、ファイル操作、API 連携など)
- 既存のコードを改良・拡張する
- 自己改善のためのコードを書く (ログ解析、統計生成、自動化スクリプトなど)
- 大きな変化や行動を選択する
- 変更理由を日本語で文章化する
- 状態を確認してから次を考える


## 制限・注意点

- 外部サービスへの課金行為は禁止
- 不要な大量通信は禁止
- 自己増殖的な挙動は禁止
